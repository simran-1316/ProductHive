# ProductHive 🛍️

ProductHive is a frontend shopping website focused on beauty products and cosmetics. The project was built to practice frontend web development concepts, responsive layouts, and user-friendly interface design using HTML, CSS, JavaScript, and Bootstrap.

## 🚀 Features

- Responsive shopping website layout
- Beauty product collection sections
- Product cards with modern UI
- Navigation bar and category sections
- Interactive frontend design

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Bootstrap

## 📚 What I Learned

Through this project, I improved my understanding of:
- Frontend web development
- Responsive web design
- Bootstrap components and layouts
- Website structuring and styling
- Interactive UI development

## 🎯 Purpose

This project was built to practice creating a modern e-commerce frontend interface and improve my frontend development skills through hands-on project building.
